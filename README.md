# yokaiLauncher
## Launcher for Minecraft
Launcher is developing on C/C++ using Qt5;

### Compilation:
```bash
mkdir build && cd build
qmake ..
make
```

Linux: `build/yokaiLauncher`
Win: `build/release/yokaiLauncher.exe`
